# การบันทึกสถานะการเปิด/ปิดคอลัมน์ในกราฟโครงสร้างทางการเงิน

## สรุปการแก้ไข

ได้ทำการปรับปรุงระบบกราฟโครงสร้างทางการเงิน (Financial Structure) ให้สามารถบันทึกสถานะการเปิด/ปิดคอลัมน์ต่างๆ เมื่อเปลี่ยนไปดูโรงพยาบาลอื่น

## คุณสมบัติใหม่

เมื่อคุณกดที่หัวคอลัมน์ของแต่ละโรงพยาบาลเพื่อเปิด/ปิด:
- **สินทรัพย์ (Assets)**
- **หนี้สิน (Liabilities)**  
- **รายได้ (Revenue)**
- **ค่าใช้จ่าย (Expenses)**
- **สถิติบริการ (Services)**
- หรือหัวข้อย่อยอื่นๆ ภายในแต่ละหมวด

สถานะเหล่านี้จะถูกบันทึกไว้ และเมื่อคุณปิดกราฟแล้วไปดูโรงพยาบาลอื่น กราฟจะเปิดหรือปิดตามที่คุณเคยตั้งค่าไว้

## การทำงาน

### ก่อนการแก้ไข
- เมื่อเปิดกราฟของโรงพยาบาล A และปิดคอลัมน์ "หนี้สิน"
- แล้วปิดกราฟไปดูโรงพยาบาล B
- กราฟจะแสดงคอลัมน์ "หนี้สิน" อีกครั้ง (กลับไปเป็นค่าเริ่มต้น)

### หลังการแก้ไข  
- เมื่อเปิดกราฟของโรงพยาบาล A และปิดคอลัมน์ "หนี้สิน"
- แล้วปิดกราฟไปดูโรงพยาบาล B
- กราฟจะยังคงปิดคอลัมน์ "หนี้สิน" ตามที่คุณตั้งค่าไว้

## รายละเอียดทางเทคนิค

### สถานะที่ถูกบันทึก

1. **window.globalFinGroupState** - บันทึกว่าแต่ละกลุ่มย่อยแสดงแบบสรุป (summary) หรือแบบละเอียด (detailed)
2. **window.globalFinSubItemState** - บันทึกว่ารายการย่อยแต่ละรายการเปิดหรือปิด
3. **window.globalFinStackState** - บันทึกว่าหมวดหลัก (Assets, Liabilities, etc.) แสดงหรือซ่อน
4. **window.globalFinSubGroupVisibility** - บันทึกการมองเห็นของกลุ่มย่อย

### การเปลี่ยนแปลงในโค้ด

ไฟล์: `Financial_Structure.html`
ฟังก์ชัน: `updateFinancialChart()`

เปลี่ยนจากการใช้ตัวแปรท้องถิ่น (local variables) เป็นตัวแปรส่วนกลาง (global variables) ที่คงอยู่ตลอดการใช้งาน

```javascript
// เดิม: สร้างใหม่ทุกครั้ง
if (!window.finGroupState) {
    window.finGroupState = {};
}

// ใหม่: ใช้ตัวแปรส่วนกลางที่คงอยู่
if (!window.globalFinGroupState) {
    window.globalFinGroupState = {};
}
window.finGroupState = window.globalFinGroupState;
```

## ประโยชน์

1. **ประสบการณ์ผู้ใช้ที่ดีขึ้น** - ไม่ต้องตั้งค่าการแสดงผลใหม่ทุกครั้งที่เปลี่ยนโรงพยาบาล
2. **ประหยัดเวลา** - เมื่อเปรียบเทียบหลายโรงพยาบาล สามารถดูข้อมูลในมุมมองเดียวกันได้ทันที
3. **ความสอดคล้อง** - การแสดงผลเหมือนกันทุกโรงพยาบาลตามที่ผู้ใช้ต้องการ

## วิธีใช้งาน

1. เปิดกราฟโครงสร้างทางการเงินของโรงพยาบาลใดก็ได้
2. คลิกที่หัวคอลัมน์เพื่อเปิด/ปิดหมวดต่างๆ ตามต้องการ
3. ปิดกราฟ
4. เปิดกราฟของโรงพยาบาลอื่น
5. กราฟจะแสดงผลตามการตั้งค่าที่คุณเลือกไว้ในขั้นตอนที่ 2
